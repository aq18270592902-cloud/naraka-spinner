# 永劫无间 · 技能随机转盘

> 千机赛季 · F1/F2 + V1/V2

纯前端的永劫无间英雄技能随机抽取转盘，支持全英雄 F 技能和 V 奥义的随机组合。

## 功能

- 🎯 转盘随机抽取英雄
- 🃏 卡牌式选择 F 技能（F1 / F2）
- 🃏 卡牌式选择 V 奥义（V1 / V2）
- ⚡ 一键随机（英雄 + F + V 全部随机）
- 📜 历史记录

## 英雄列表（16位）

宁红夜、特木尔、迦南、季沧海、胡桃、天海、妖刀姬、崔三娘、岳山、无尘、顾清寒、武田信忠、殷紫萍、沈妙、胡为、甘璇

## 本地运行

```bash
node server.js
# 访问 http://localhost:8090
```

## 技术栈

- 纯 HTML + CSS + JavaScript（无框架）
- Canvas API 绘制转盘
- Node.js 静态文件服务器
